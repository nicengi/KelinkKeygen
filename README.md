# KelinkKeygen
## 使用方法
1. 将 kelink_keygen 文件夹中的文件覆盖到原程序的bin文件夹中。
2. 运行脚本生成新注册码，填入到 Web.config 中的 KL_License 字段。

## 介绍
整理硬盘的时候挖出了这个东西->“柯林WAP建站程序最新破解版 V12.08.11.zip”，想起来很久以前研究过，随即在虚拟机里把它部署了一下，发现里面的注册码绑定的域名是无法修改的，研究了一下。这应该是10年代的老东西了，开发它的公司已经没有了。用的是 ASP.NET+MSSQL。
